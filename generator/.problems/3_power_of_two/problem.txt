Given an integer n, return 1 if it is a power of two. Otherwise, return 0.

An integer n is a power of two, if there exists an integer x such that n == 2^x.

 

Example 1:

Input: n = 1
Output: 1
Explanation: 2^0 = 1

Example 2:

Input: n = 16
Output: 1
Explanation: 2^4 = 16

Example 3:

Input: n = 3
Output: 0

 

Constraints:

    -231 <= n <= 231 - 1
